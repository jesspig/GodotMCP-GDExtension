# v0.1.1 — 功能里程碑

**发布日期**：2026-05-24

大规模工具扩展和 CMake 构建系统迁移。

### 新增

- **14 个新工具命令**：
  - 节点操作（node.rs）：创建/删除/重命名/复制/移动/重设父节点
  - 场景操作（scene.rs）：打开/保存/创建/删除场景
  - 属性操作（property.rs / property_3d.rs）：2D/3D 属性读写
  - 碰撞体（collision.rs）：圆形/矩形碰撞体添加
  - 查找（find.rs）：按名称/类型/组/脚本搜索节点
  - 脚本管理（script_gd.rs / script_cs.rs）：GDScript 和 C# 脚本创建/编辑
  - 脚本辅助（script_helpers.rs）：call_method、get/set_variable
  - 搜索（search.rs）：文件内搜索、项目全文搜索、查找替换
  - 项目设置（project_settings.rs）：读取/写入项目设置
  - 撤销/重做（undo.rs）
- **LSP 客户端** —— `lsp/client.rs`，通过 StreamPeerTCP 连接 Godot LSP
- **UndoRedo 基础设施**
- **多语言 Wiki 文档**（中文 + 英文）

### 重构

- **CMake 构建系统** —— 替代旧 `package_addons.py` 脚本
- **命令模块拆分** —— 从 `node.rs` 拆分出属性、查找、编辑器控制命令
- **项目目录调整** —— 插件移至 `godot/addons/`
- **工具注册表扩展** —— 支持新工具注册

### 修复

- 完善命令实现鲁棒性
