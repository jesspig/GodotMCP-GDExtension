# v0.1.5 — GDExtension 纯 C++ 重写

**发布日期**：2026-05-30

移除 Rust 和 Python 组件，全面迁移至 C++ GDExtension。

### 重大变更

- **移除 Rust 遗留代码** —— 删除 `crates/core`、`crates/gdext`、`crates/server` 所有 Rust crate
- **移除 Python MCP 服务器** —— 删除 `server/` 目录及其测试套件
- **移除规划文档** —— 删除 `docs/plan/` 目录

### 新增

- **纯 C++ GDExtension 实现** —— `extensions/gdext/` 完全替换旧 Rust + Python 架构
  - HTTP 服务器（`ipc/http_server.cpp`）—— 零依赖手动 HTTP/1.1
  - MCP 处理器（`mcp/mcp_handler.cpp`）—— JSON-RPC 2.0 + SSE
  - 处理器注册表（`commands/handler_registry.cpp`）—— 函数指针路由
- **工具 Schema 系统** —— `tools/tool_schemas.json`（1977 行），构建时复制到输出目录
- **122 个工具全部用 C++ 重写** —— 覆盖场景、节点、属性、脚本、搜索、碰撞、编辑器控制、项目设置、输入映射、插件管理、Autoload
- **LSP 客户端** —— C++ 版 `lsp/client.cpp`，通过 StreamPeerTCP 连接 Godot LSP
- **工具函数**（`cmd_utils.cpp`）—— `resolve_node`、`undoable_set`、JSON 工具链

### 重构

- **单进程架构** —— 移除双进程（GDExtension + Python Server）架构，所有逻辑在 GDExtension 内完成
- **Streamable HTTP 传输** —— 替换 WebSocket 传输
- **构建系统精简** —— 纯 CMake + `build.py` Python 封装，移除 Rust/Cargo 工具链
- **CI/CD 更新** —— 跨平台 CMake 构建（Ubuntu/MacOS/Windows），Tag 即发布

### 修复

- 场景/节点/脚本操作中的多个 Bug
- HTTP 服务器健壮性增强
- 移除 WebSocket 服务器遗留代码
