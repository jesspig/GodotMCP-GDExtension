# 运行时工具

14 个用于控制游戏运行时和桥接游戏进程的工具。

## 运行时生命周期（6 个）

### `run_project`

**分类**: `runtime_tools/lifecycle`  
**描述**: 运行项目。  
**破坏性**: 否 | **撤销**: 否

### `run_current_scene`

**分类**: `runtime_tools/lifecycle`  
**描述**: 运行当前场景。  
**破坏性**: 否 | **撤销**: 否

### `run_specific_scene`

**分类**: `runtime_tools/lifecycle`  
**描述**: 运行指定的场景文件。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `scene_path` | `string` | 是 | 场景文件路径 |

### `stop_project`

**分类**: `runtime_tools/lifecycle`  
**描述**: 停止运行中的项目。  
**破坏性**: 否 | **撤销**: 否

### `pause_project`

**分类**: `runtime_tools/lifecycle`  
**描述**: 暂停或恢复运行中的项目。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `paused` | `bool` | 否 | true = 暂停, false = 恢复 |

### `set_movie_maker`

**分类**: `runtime_tools/lifecycle`  
**描述**: 切换 Movie Maker 模式。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `enabled` | `bool` | 是 | 启用或禁用 movie maker |

## 运行时桥接（8 个）

### `wait_for_bridge`

**分类**: `runtime_tools/bridge`  
**描述**: 等待运行时桥接连接。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 默认值 | 说明 |
|------|------|----------|---------|-------------|
| `timeout_ms` | `int` | 否 | 5000 | 最大等待时间（毫秒） |

### `get_game_scene_tree`

**分类**: `runtime_tools/bridge`  
**描述**: 获取游戏运行时的场景树。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 默认值 | 说明 |
|------|------|----------|---------|-------------|
| `timeout_ms` | `int` | 否 | 5000 | 桥接超时时间 |

### `get_game_node_property`

**分类**: `runtime_tools/bridge`  
**描述**: 获取运行游戏中节点的属性。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `node_path` | `string` | 是 | 游戏中的节点路径 |
| `property` | `string` | 是 | 属性名称 |
| `timeout_ms` | `int` | 否 | 桥接超时时间（默认: 5000） |

### `set_game_node_property`

**分类**: `runtime_tools/bridge`  
**描述**: 设置运行游戏中节点的属性。  
**破坏性**: 是 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `node_path` | `string` | 是 | 游戏中的节点路径 |
| `property` | `string` | 是 | 属性名称 |
| `value` | variant | 是 | 新值 |
| `timeout_ms` | `int` | 否 | 桥接超时时间（默认: 5000） |

### `call_method_in_game`

**分类**: `runtime_tools/bridge`  
**描述**: 在运行游戏中调用方法。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `node_path` | `string` | 是 | 游戏中的节点路径 |
| `method` | `string` | 是 | 方法名称 |
| `args` | `array` | 否 | 方法参数 |
| `timeout_ms` | `int` | 否 | 桥接超时时间（默认: 5000） |

### `capture_game_screenshot`

**分类**: `runtime_tools/bridge`  
**描述**: 从运行游戏中截取屏幕截图。  
**破坏性**: 否 | **撤销**: 否

### `simulate_game_input`

**分类**: `runtime_tools/bridge`  
**描述**: 在运行游戏中模拟输入。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `action` | `string` | 是 | 输入动作名称 |
| `pressed` | `bool` | 是 | 按键按下状态 |
| `timeout_ms` | `int` | 否 | 桥接超时时间（默认: 5000） |

### `list_game_instances`

**分类**: `runtime_tools/bridge`  
**描述**: 列出所有已连接的运行游戏实例。  
**破坏性**: 否 | **撤销**: 否

**返回值**: 返回一个数组，包含所有当前已连接的运行游戏实例及其 ID、连接时间戳和运行时长。|
