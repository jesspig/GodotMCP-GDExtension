# 信号与分组工具

8 个用于管理节点信号和分组的工具。

## 信号管理（4 个）

### `connect_signal`

**分类**: `node_tools/signal`  
**描述**: 将节点信号连接到方法。  
**破坏性**: 否 | **撤销**: 否 | **需要节点**: 是

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `node_path` | `string` | 是 | 节点路径 |
| `signal_name` | `string` | 是 | 信号名称 |
| `target_path` | `string` | 是 | 目标节点路径 |
| `method_name` | `string` | 是 | 目标方法名称 |
| `flags` | `int` | 否 | 连接标志 |

### `disconnect_signal`

**分类**: `node_tools/signal`  
**描述**: 断开节点信号。  
**破坏性**: 否 | **撤销**: 否 | **需要节点**: 是

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `node_path` | `string` | 是 | 节点路径 |
| `signal_name` | `string` | 是 | 信号名称 |
| `target_path` | `string` | 是 | 目标节点路径 |

### `list_signals`

**分类**: `node_tools/signal`  
**描述**: 列出节点的所有信号。  
**破坏性**: 否 | **撤销**: 否 | **需要节点**: 是

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `node_path` | `string` | 是 | 节点路径 |

### `get_signal_connections`

**分类**: `node_tools/signal`  
**描述**: 获取信号连接列表。  
**破坏性**: 否 | **撤销**: 否 | **需要节点**: 是

## 节点分组（4 个）

### `add_to_group`

**分类**: `node_tools/group`  
**描述**: 将节点添加到分组。  
**破坏性**: 否 | **撤销**: 是 | **需要节点**: 是

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `node_path` | `string` | 是 | 节点路径 |
| `group_name` | `string` | 是 | 分组名称 |

### `remove_from_group`

**分类**: `node_tools/group`  
**描述**: 将节点从分组移除。

**描述**: 获取节点所属的分组。

#### 参数

| 名称 | 类型 | 必需 | 说明 |

| `group_name` | `string` | 是 | 分组名称 |
