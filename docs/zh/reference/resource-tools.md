# 资源与通用兜底工具

8 个用于管理资源和通用属性访问的工具。

## 资源操作（6 个）

### `save_resource`

**分类**: `node_tools/general`  
**描述**: 将资源保存到文件。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `path` | `string` | 是 | 资源路径 |
| `resource_path` | `string` | 是 | 源资源路径 |

### `load_resource`

**分类**: `node_tools/general`  
**描述**: 从路径加载资源。  
**破坏性**: 否 | **撤销**: 否

#### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `path` | `string` | 是 | 要加载的路径 |

### `new_resource`

**分类**: `node_tools/general`

| `type` | `string` | 是 | 资源类名 |
