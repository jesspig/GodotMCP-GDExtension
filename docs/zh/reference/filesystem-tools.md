# 文件系统工具

12 个用于创建、删除和管理项目文件的工具。

## `create`

**分类**: `editor_tools/filesystem`  
**描述**: 创建文件。  
**破坏性**: 否 | **撤销**: 否

### 参数

| 名称 | 类型 | 必需 | 说明 |
|------|------|----------|-------------|
| `path` | `string` | 是 | 文件路径 |
| `content` | `string` | 否 | 文件内容 |
