# Project Structure

## Top-Level Layout

```
GodotMCP/
extensions/              C++ GDExtension source
  CMakeLists.txt         GDExtension build (godot-cpp 10.0.0-rc1)
  src/                   All C++ source code
docs/                    Documentation source (Rspress)
  en/                    English docs
  zh/                    Chinese docs
  public/                Static assets
doc_build/               Built documentation output
example/                 Godot test project
  addons/godot_mcp/      Plugin files (auto-generated by build)
tests/                   YAML test framework
  yaml_tests/            26 YAML test files
  test_orchestrator.py   Python test orchestrator
  godot_manager.py       Godot process lifecycle
scripts/                 Python build/package scripts
cmake/                   CMake modules (8 files)
main.py                  Build/test/package entry point
rspress.config.ts        Documentation site config
CMakeLists.txt           Root CMake build (version: 0.2.1)
```

## Source Code Layout (extensions/src/)

```
extensions/src/
register_types.cpp          GDExtension entry (symbol: gdext_mcp_init)
editor_plugin.cpp/.hpp      EditorPlugin lifecycle + _process() pump
client_config_registry.hpp  MCP client config templates (11 providers)

sdk/
  mcp_tool_definition       SDK base class (GDScript/C# inheritable)
  mcp_tool_registry         Engine singleton registry

server/
  ipc/                      HTTP server layer
  mcp/                      MCP protocol handler (no sessions)
  registry/                 HandlerRegistry (tool dispatch + search)

built_in/
  tool_base                 ITool base class + type validation
  tool_adapter              SDK to ITool adapter
  cmd_utils                 Shared utilities
  cmd_utils_json            JSON to Variant conversion
  cmd_utils/                7 template headers
  register_itools.cpp       X-macro registration master
  tools/                    153 tool implementations
    meta/                   7 meta tools
    signal/                 4 signal tools
    group/                  4 group tools
    node_tools/             6 resource + 2 fallback
    editor_tools/           19 subcategories
    runtime_tools/          7 bridge + 6 lifecycle

runtime/
  bridge                    Editor-side TCP client
  game_bridge               Game process TCP server

ui/
  mcp_dock                  Editor right panel dock
  mcp_logger                C++ structured logger
  mcp_console               Editor output console

testing/
  test_engine               YAML pipeline test engine
  pipeline_parser           Pipeline YAML parser
  pipeline_runner           Pipeline executor
  pipeline_context          Pipeline context
  pipeline_types            Type definitions
  yaml_parser               YAML parsing utilities
  test_assertions           Assertion helpers
  godot_file_verifier       Disk verification
```
