# v0.1.1 — Feature Milestone

**Release Date**: 2026-05-24

Major tool expansion and CMake build system migration.

### New Features

- **14 new tool commands**:
  - Node operations (node.rs): create/delete/rename/duplicate/move/reparent
  - Scene operations (scene.rs): open/save/create/delete scenes
  - Property operations (property.rs / property_3d.rs): 2D/3D property read/write
  - Collision shapes (collision.rs): circle/rectangle collision shape creation
  - Search (find.rs): find nodes by name/type/group/script
  - Script management (script_gd.rs / script_cs.rs): GDScript and C# script create/edit
  - Script helpers (script_helpers.rs): call_method, get/set_variable
  - Search (search.rs): in-file search, full-text project search, find/replace
  - Project settings (project_settings.rs): read/write project settings
  - Undo/Redo (undo.rs)
- **LSP client** — `lsp/client.rs`, connects to Godot LSP via StreamPeerTCP
- **UndoRedo infrastructure**
- **Multi-language Wiki documentation** (Chinese + English)

### Refactoring

- **CMake build system** — replaced old `package_addons.py` script
- **Command module splitting** — property, search, and editor control commands split from `node.rs`
- **Project directory adjustments** — plugin moved to `godot/addons/`
- **Tool registry extension** — supports new tool registration

### Fixes

- Improved command implementation robustness
