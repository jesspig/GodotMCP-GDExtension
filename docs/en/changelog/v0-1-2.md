# v0.1.2 — Editor Tool Expansion

**Release Date**: 2026-05-24

Added editor control, input mapping, plugin management, and other tool groups.

### New Features

- **Editor control commands** — `editor_control.rs`: play/stop, refresh filesystem
- **Input mapping commands** — `input_map.rs`: add/remove/set input actions
- **Plugin management commands** — `plugin_management.rs`: enable/disable plugins
- **Project settings extension commands** — `project_settings_ext.rs`: display/physics/rendering settings

### Enhancements

- **Collision command enhancements** — UndoRedo support
- **GDScript commands** — validate, list_gdscripts improvements

### Refactoring

- **Project structure reorganization** — `godot/` renamed to `example/`
- **Wiki directory flattening** — removed multi-language (en/zh) directories, kept Chinese only
- **README file renaming** — `README.zh-CN.md` → `README-zh.md`

### Documentation

- Restructured Wiki documentation
- Updated README and AGENTS.md
