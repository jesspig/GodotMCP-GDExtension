## 2. PERT 三点估算

### 2.1 PERT 公式

```
期望时间 (TE) = (O + 4M + P) / 6
标准差 (σ) = (P - O) / 6
方差 (σ²) = ((P - O) / 6)²

O = 乐观估算 (Optimistic)
M = 最可能估算 (Most Likely)
P = 悲观估算 (Pessimistic)
```

## 6. 储备计算

### 6.1 应急储备 (Contingency Reserve)

基于 PERT 方差计算：

```
关键路径方差 σ² = 0.25 (H) + 0.44 (I) + 0.11 (K) + 0.25 (测试) = 1.05
关键路径标准差 σ = 1.02d

应急储备 = 2σ = 2.04d ≈ 2d
```

### 6.2 管理储备 (Management Reserve)

```
管理储备 = 项目基础工期 × 15% = 12.84d × 15% = 1.93d ≈ 5d
（取整并考虑未识别风险的不确定性）
```

### 6.3 总储备

| 储备类型 | 大小 | 占比 | 用途 |
|---------|------|------|------|
| 应急储备 | 2d | 15.6% | 已识别风险（R1-R10） |
| 管理储备 | 5d | 38.9% | 未识别风险、范围变更 |
| **总储备** | **7d** | **54.5%** | |

### 6.4 储备燃尽图计划

```
Sprint 1 (3d):  应急 0.5d + 管理 1d = 1.5d 储备
Sprint 2 (4d):  应急 0.5d + 管理 1.5d = 2d 储备
Sprint 3 (9d):  应急 1d + 管理 2.5d = 3.5d 储备
总计: 7d 储备
```

**触发规则**：
- 应急储备：由主 Agent 根据风险登记表中的触发条件决定使用
- 管理储备：需明确记录使用原因，用于未识别风险
- 储备使用 >50% 时触发项目状态审查
